
## Description

[Upgradable Contracts](https://github.com/bogdan-manole/upgradable-contracts-example)

## Installation

```bash
$ npm install
```

## Container
```
# to deploy
npx aeproject env init

# to stop container
npx aeproject env --stop

```

## Running the tests

```bash
# tests
$ npm run test

# debug mode
$ npm run test:debug

```

## Deploy

```bash
# deploy on local node the factory contract
$ npm run deploy

```
